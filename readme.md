# bitmania

## 概要
音楽ゲームシュミレーター  
https://masan-k.github.io/bitmania/www/index.html


## 前提
入力は8つ（7鍵盤＋皿を想定）

## ゲームモード(検討中)
* demo
キーボード入力結果を画面に表示する

* stairs challenge
制限時間以内にできるだけ多く階段を入力する
* rhythm patterm
決まったリズムパターンをテンポに合わせて入力する

## メニューオプション
* キーボード設定：入力1~8をキーボードで設定

## ゲームモードオプション
* stairs challenge
- 制限時間：10、30、60
* rhythm patterm
- リズムパターンの選択：8ビート、16ビート、シャッフル
- テンポの選択：60bpm~300bpm
